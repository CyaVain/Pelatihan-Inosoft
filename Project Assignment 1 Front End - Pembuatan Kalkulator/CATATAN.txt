#Sengaja tidak dibuat responsive karena terlihat tidak enak dilihat jika tanpa menggunakan frame hp

#Program dapat menerima INPUT KEYBOARD BERUPA ANGKA 0-9, dan simbol: 
bintang (*) atau X (x) untuk perkalian,
divide (/) untuk pembagian,
persen (%) untuk konversi menjadi persen yang berupa desimal seperti kalkulator pada umumnya, 
plus (+) untuk penambahan dan strip (-) untuk pengurangan,
desimal atau titik (.) untuk mengkonversi angka menjadi nilai desimal.

Cara input via keyboard : 
> klik pada kolom Input area yang memiliki shadow box 
> ketikan angka 
> ketik operasi yang diinginkan lewat keyboard atau tekan tombol pada layar untuk operasi  
> tekan Enter pada keyboard
> hasil akan muncul pada History

silahkan lanjutkan penggunaan kalkulator seperti biasa,

jika History sudah dirasa kebanyakan, bisa dihapus.

untuk menghapus History, tekan tombol AC
tombol AC akan berubah jadi C apabila Input area memiliki isi didalamnya,
C hanya untuk menghapus seluruh Input saja,
bisa gunakan tombol Backspace yang berada pada sebelah tombol 0, untuk menghapus per digit (1 digit) setiap di tekan, 
tombol AC akan muncul apabila kolom input sudah kosong,
